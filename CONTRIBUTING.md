# Contributing to Omniperf

Thank you for your interest in contributing to the Isaac Lab Performance Dashboard!

## Developer Certificate of Origin (DCO)

All contributions to this project must be accompanied by a
[Developer Certificate of Origin](https://developercertificate.org/) (DCO) sign-off.
This certifies that you wrote (or otherwise have the right to submit) your contribution
under the project's open-source license.

To sign off on your commits, add a `Signed-off-by` line to your commit message:

```
Signed-off-by: Your Name <your.email@example.com>
```

You can do this automatically by using the `-s` flag when committing:

```bash
git commit -s -m "Your commit message"
```

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/my-change`)
3. Make your changes
4. Commit with DCO sign-off (`git commit -s -m "Description of change"`)
5. Push to your fork (`git push origin feature/my-change`)
6. Open a Pull Request

## Reporting Issues

Please use [GitHub Issues](https://github.com/NVIDIA/omniperf/issues) to report bugs or request features.
